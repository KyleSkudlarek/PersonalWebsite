About Me

I'm Kyle Skudlarek, a software engineer and freelance web developer. I've been working with computers since I was a kid, helping neighbors with various IT issues and building my first computer for gaming when I was 12. I graduated from the University of Virginia in 2015 with a Bachelor of Science in Computer Science. Since then I've been working in the industry as a software engineer. I have extensive experience as a backend engineer building building web apps, APIs, and microservices in AWS. I'm currently learning the frontend, with a basic understanding of html, css and javascript.
